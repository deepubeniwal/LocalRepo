Git & Github

Git

Git is a version control system is a tools that helps to tracks changes in code.

Git is a version control system.

It is very popular, free and open source with fast and scalable community.

Work of Git

  • 1. Track the history of a code.
  • 2. Store multiple versions of a file or directory.
  • 3. Allow users to collaborate on projects by sharing files and allowing others to make edits to those files.

Github

Github is the largest social coding and collaboration site where programmers can share their codes with others.

Github is an online service and web based Git repository hosting service.
